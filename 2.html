html tag 설명

안녕하세요. 오늘은 html의 기본적인 tag들에 대해서 배워보겠습니다.
먼저 h1 tag 는 heading의 약자로, 제목을 표현하고 싶을때 사용합니다.
글자를 진하게 강조하고 싶을때는 strong tag를 사용하구요.
밑줄을 그어서 특정 부분을 강조하고 싶을때는 u tag를 사용합니다.
p태그는 paragraph 단락을 나타내기 위해 사용하는 tag 입니다.
마지막으로 br tag는 줄바꿈을 위해 사용하는 tag인데요.
다른 tag들은 n으로 시작해 /n으로 끝내서 어디서부터 어디까지가 태그를 사용하는 부분인지 표시를 해주었습니다만,
br태그는 구역이 아니라 특정 부분에서 줄바꿈을 하면 되는것이기 떄문에 /br로 시작과 끝을 따로 정하지 않습니다.
